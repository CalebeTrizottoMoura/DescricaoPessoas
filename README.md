# Dados Pessoais

Este é um programa simples em C# que solicita informações básicas de uma pessoa e exibe algumas informações sobre ela.

## Funcionalidades

O programa realiza as seguintes tarefas:

- Solicita o nome da pessoa, verificando se é um nome válido;
- Solicita a data de nascimento no formato DD/MM/AAAA, validando a entrada;
- Solicita a altura da pessoa, validando a entrada;
- Solicita o endereço da pessoa;
- Calcula a idade da pessoa com base na data atual;
- Exibe na tela o nome, data de nascimento, idade, altura e endereço da pessoa.

## Como Executar

Para executar o programa:

1. Certifique-se de ter o ambiente de desenvolvimento C# configurado.
2. Compile o programa utilizando um compilador de C# de sua escolha.
3. Execute o arquivo compilado.

- Exemplo de execução no terminal: dotnet run

## Requisitos

- Ambiente de desenvolvimento C# configurado.
